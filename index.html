NayFit

Este proyecto es una landing page para Nayla Veron, una entrenadora personal. La página presenta los diversos planes de entrenamiento que ofrece, incluyendo sesiones personalizadas, entrenamiento en casa y a distancia. Utilicé tecnologías modernas para crear una experiencia de usuario atractiva y responsiva. El objetivo principal es proporcionar información clara y accesible sobre los servicios de Nayla, fomentando el contacto a través de la plataforma

Helanus

Este proyecto consiste en un backend para el e-commerce de Helanus, una heladería artesanal. Utilizando MongoDB para la base de datos, la plataforma permite gestionar productos, pedidos y usuarios de manera eficiente. Implementé funciones clave para la experiencia del cliente, como la visualización de productos, la gestión del carrito de compras y la realización de pedidos. Este sistema asegura un manejo robusto y escalable de los datos, facilitando las operaciones diarias de la heladería

Strong Suplementos

Este proyecto es un e-commerce para una tienda de suplementos deportivos, desarrollado con React y Firebase. La plataforma permite a los usuarios explorar una variedad de productos, agregar artículos al carrito y realizar compras. Utilicé Firebase para gestionar la base de datos, asegurando una experiencia fluida y eficiente. La interfaz moderna y responsiva facilita la navegación y mejora la interacción del usuario.

Bikes Dynamo

Este proyecto es una simulación de un e-commerce para una tienda de bicicletas, desarrollada con tecnologías web modernas. La plataforma permite a los usuarios explorar diferentes modelos de bicicletas, conocer sus características y realizar compras simuladas. El sitio está diseñado para proporcionar una experiencia de usuario fluida y atractiva, con un enfoque en la navegación intuitiva y la presentación clara de los productos.
